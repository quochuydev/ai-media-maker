# FAQ

Frequently asked questions about AI Media Maker.

## General

### What is AI Media Maker?

AI Media Maker is an AI-powered image generation platform. Enter text prompts and get high-quality images generated by advanced AI models. Supports bulk generation of up to 10 images per batch.

### How good are the generated images?

Images are generated at 1024x1024 resolution using state-of-the-art AI models. Quality depends on the specificity and clarity of your prompts.

### What kind of images can I generate?

You can generate any kind of image from a text description — landscapes, portraits, abstract art, product mockups, illustrations, and more.

## API & Billing

### How do API credits work?

Each image generation consumes 1 credit. You can purchase credits in the [Billing](/c/cloud/billing) section.

### What payment methods do you accept?

We accept PayPal for all credit purchases.

### Is there a free tier?

New users get 3 free credits to try the service. The [Playground](/playground) requires credits to use.

### How do I get an API key?

1. Sign in at [maker.cappuai.com/login](/login)
2. Go to [API Keys](/c/cloud/api-keys)
3. Click "Add Key" and copy your key

API keys use the `wi_live_` prefix and are hashed for security.

## Security

### How is my data handled?

- Prompts are processed and immediately discarded
- Generated images are stored securely on Cloudflare R2
- API keys are hashed using SHA-256 before storage
- All traffic is encrypted using TLS 1.3

### Where are images stored?

Images are stored on Cloudflare R2, a globally distributed object storage service, and are accessible via secure URLs.

## Technical

### What is the API rate limit?

Contact us for rate limit details based on your usage needs.

### How many images can I generate at once?

Up to 10 images per batch in the playground. API users can make multiple batch requests.

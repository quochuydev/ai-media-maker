# Overview

AI Media Maker is an AI-powered image generation platform that lets you create stunning images from text prompts with bulk generation and cloud storage.

## Features

- **AI Image Generation** - Generate high-quality images from text prompts using advanced AI models
- **Bulk Generation** - Create up to 10 images at once with different prompts
- **Cloud Storage** - Images stored securely on Cloudflare R2
- **Credit-based Billing** - Pay only for what you use
